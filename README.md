# Класифікація текстових запитів у сфері туризму (NLP dataset)

Цей репозиторій містить користувацький набір текстових даних (`dataset_180.csv`), зібраний та розмічений спеціально для задач обробки природної мови українською мовою. Датасет орієнтований на розпізнавання намірів (intent classification) користувачів у туристичній сфері.

## 📂 Структура даних
Файл у форматі CSV містить три колонки:

| Назва колонки | Тип даних | Опис |
| :--- | :--- | :--- |
| `text` | String | Текст запиту або відгуку клієнта українською мовою. |
| `label_number` | Integer | Числовий ідентифікатор класу (від 0 до 5). |
| `label_name` | String | Текстова назва категорії (інтенту). |

## Класифікація інтентів 
Датасет розмічено на 6 категорій, які відповідають основним етапам взаємодії клієнта з туристичним сервісом:

*   **0 — Підбір туру:** Запити щодо пошуку напрямків, типів відпочинку та умов подорожі.
*   **1 — Ціни на подорожі:** Питання щодо вартості турів, додаткових послуг, наявності знижок та фінансових умов.
*   **2 — Бронювання:** Запити, пов'язані з процесом резервування, внесенням передоплати, підтвердженням броні та умовами її скасування.
*   **3 — Гіди і супровід:** Питання про наявність екскурсоводів, рівень їхньої кваліфікації, знання мов та організацію індивідуальних екскурсій.
*   **4 — Зміна дат туру:** Запити на перенесення дати подорожі, інформація про штрафні санкції та правила внесення змін до договору.
*   **5 — Відгуки про напрямки:** Емоційно забарвлені коментарі (як позитивні, так і негативні) про досвід відвідування різних країн та міст.

## Можливі сценарії використання
Цей набір даних створено **виключно в навчальних та дослідницьких цілях**. Він ідеально підходить для:
1. навчання та тестування моделей класифікації тексту (text classification) у рамках студентських або особистих проєктів.
2. розробки та тренування прототипів туристичних чат-ботів для портфоліо.
3. аналізу тональності тексту (sentiment analysis) на базі клієнтських відгуків.
